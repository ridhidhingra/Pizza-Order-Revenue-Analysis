-) Project Overview

This project analyzes pizza store sales data using SQL to uncover key business insights related to orders,
revenue, product performance, and customer demand. The dataset represents a real-world food business 
scenario involving orders, pizzas, and pizza categories.

-) Objectives

Calculate total orders and total revenue

Identify highest-priced and most popular pizzas

Analyze pizza size and category-wise demand

Find top-selling and top revenue-generating pizzas

Study order distribution by hour and daily demand trends

-) Key Analysis Performed

Total number of orders placed

Total revenue generated

Most common pizza size ordered

Top pizza types based on quantity and revenue

Category-wise sales and revenue contribution

Hourly order distribution

Average number of pizzas ordered per day

-) Tools & Skills Used

SQL

Joins (INNER JOIN)

Aggregations (SUM, COUNT, AVG)

Group By & Order By

Subqueries

Date & Time functions

-)Dataset Tables

orders

order_details

pizzas

pizza_types


-) Conclusion

This project demonstrates strong SQL fundamentals and analytical thinking by solving real-world
business questions. It is ideal for showcasing SQL skills in a data analyst portfolio.
